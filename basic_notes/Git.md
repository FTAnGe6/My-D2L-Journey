# GitHub 及软件方面notes
## Git状态缩写
### U (Untracked)：未跟踪。

- 意思：这是一个全新的文件，Git 以前从来没见过它。

- 现状：它只存在于你的磁盘上，还没进入 Git 的管理范围。如果你现在提交，它是不会被包含进去的。

- 如何消除：点击文件旁的 “+”号（暂存），它就会变成 A (Added)，表示已准备好第一次入库。

### M (Modified)：已修改。

- 意思：这个文件 Git 之前已经记录过了，但你刚刚又改动了内容。

- 现状：本地的内容比仓库里的内容要新。

- 如何消除：点击提交（Commit）后，这个字母就会消失，表示本地和仓库同步了。
